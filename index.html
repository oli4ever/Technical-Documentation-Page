Introduction

HTML5 is the latest version of Hypertext Markup Language (HTML), designed to improve the language with support for the latest multimedia while keeping it easily readable by humans and consistently understood by computers.

Introduced in 2008, HTML5 provides new elements and attributes, enhanced multimedia support, and better performance.

Its aim is to reduce reliance on plugins and external libraries for multimedia content.

<!DOCTYPE html>: Defines that this document is an HTML5 document.
  • Released in 2014 as a W3C Recommendation.
  • Enhanced semantic structure.
  • Improved multimedia support.
New Elements

HTML5 introduces several new elements to enhance functionality and semantics:

  • <article>: Defines self-contained content.
  • <section>: Represents sections of a document.
  • <header> and <footer>: Define header and footer content.
  • <aside>: Represents content related to the main content.
  • <main>: Specifies the main content area of a document.

These elements provide a clear structure and better accessibility for web pages.

Semantic Tags

Semantic tags are an important feature of HTML5, providing meaning to the content they enclose.

Common semantic tags include:

  • <nav>: Defines navigation links.
  • <figure>: Represents self-contained content, like images with captions.
  • <time>: Encodes date or time information.

For example:

<nav><a href="#section">Go to Section</a></nav>

Using semantic tags improves search engine optimization (SEO) and accessibility.

APIs

HTML5 includes a variety of APIs that enhance the user experience:

  • Canvas API: Allows drawing graphics and animations directly in the browser.
  • Geolocation API: Provides geographic location of the user.
  • Web Storage API: Offers local and session storage for data persistence.

Example usage of Canvas API:

let canvas = document.getElementById('myCanvas').getContext('2d');

These APIs reduce the dependency on third-party libraries.

Forms

HTML5 introduces new input types and attributes for forms, simplifying user input.

New input types include:

  • date: Allows selecting a date.
  • email: Validates email addresses.
  • range: Creates sliders for numerical input.

For example:

<input type="email" placeholder="Enter your email">

These features improve user interaction and validation without additional scripting.
